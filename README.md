# 📚 Assignment Tracker Flutter App

A simple, fast, and free mobile application built with Flutter to help students manage their assignments, track deadlines, and improve productivity.

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)

## ✨ Features
*   **Create Assignments:** Add assignment title, description, subject, and due date.
*   **Deadline Tracker:** Visual indicator for upcoming or overdue assignments.
*   **Status Updates:** Mark assignments as "Pending", "In Progress", or "Completed".
*   **Local Storage:** Uses SQLite/Hive to keep data local and free (no backend required).
*   **Clean UI:** Simple, intuitive interface to reduce distraction.

## 🚀 Getting Started

### Prerequisites
*   [Flutter SDK](https://docs.flutter.dev/get-started/install)
*   [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/)
*   A mobile emulator or a physical device.

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/assignment-tracker-flutter.git
    ```
2.  **Navigate to the folder:**
    ```bash
    cd assignment-tracker-flutter
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```

## 🛠️ Technologies Used
*   **Framework:** Flutter
*   **Language:** Dart
*   **Database:** Hive (fast local database) or Sqflite
*   **State Management:** Provider/Riverpod

## 📱 Usage
1.  Open the app.
2.  Tap the **"+" button** to add a new assignment.
3.  Fill in the details (Title, Subject, Due Date).
4.  Tap **Save**.
5.  Swipe left to delete or tap an assignment to mark it as complete.
